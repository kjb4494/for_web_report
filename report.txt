001. area: 이미지맵에서 영역을 지정
- 속성
	01. alt: 이미지를 사용할 수 없을 때 사용할 대체 텍스트
	02. coords: 이미지 맵에서 만들어 질 모양의 좌표
	03. shape: 이미지 맵에 생성 될 모양의 종류
	04. href: 하이퍼 링크의 주소
	05. target: 하이퍼 링크 탐색을 위한 브라우징 컨텍스트
	06. download: 리소스로 이동하지 않고 리소스를 다운로드할지의 여부와 리소스가 있는 경우 파일 이름을 다운로드할지의 여부
	07. rel: 현재 문서와 url의 관계. alternate, author, help 등이 있음
	08. hreflang: 링크된 리소스의 언어 코드
	09. type: 참조되는 리소스의 미디어 유형
	10. referrerpolicy: 라퍼러 정책 전달
- 예시
    <img src="planets.gif" width="145" height="126" alt="Planets" usemap="#planetmap">
    <map name="planetmap" id="maptest">
    <area shape="rect" coords="10,8,50,292" alt="test" href="test.html" target="_blank">
    <area shape="rect" coords="0,0,82,126" alt="Sun" href="sun.htm" rel="alternate" hreflang="en" download="sun">
    <area shape="circle" coords="90,58,3" alt="Moon" href="/images/sun.gif" type="image/gif">
    </map>

004. bdi: 텍스트의 방향성을 자동으로 제어. 즉, 히브리어, 아랍어 등 오른쪽에서 왼쪽으로 흐르는 언어 군이 필요할 때 사용
- 속성: 없음
- 예시
    <ul>
    ${ for text in texts }
    <li><bdi>${text.content}</bdi></li>
    ${ endfor }

- 예시

006. canvas: 스크립트를 이용해 즉석해서 그래픽을 그리는 데 사용
- 속성
	01. width: 너비
	02. height: 높이
- 예시
    <canvas id="myCanvas">이 브라우저는 html5 canvas 태그를 지원하지않습니다.</canvas>
    <script>
    var c = document.getElementById("myCanvas");
    var ctx = c.getContext("2d");
    ctx.fillStyle = "#FF0000";
    ctx.fillRect(0, 0, 800, 100);
    </script>
